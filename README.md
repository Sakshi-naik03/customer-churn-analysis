\# 📊 **Customer Churn Analysis using SQL \& Python**



An end-to-end Data Analysis project focused on understanding customer churn in a telecommunications company through SQL, Python, Exploratory Data Analysis (EDA), and business-driven insights.





\## 📖 **Project Overview**



Customer churn is one of the most critical business challenges faced by telecommunication companies. Losing existing customers directly impacts revenue and increases customer acquisition costs.



This project analyzes customer behavior to identify the key factors contributing to customer churn. Using SQL for querying and Python for data analysis and visualization, the project uncovers actionable business insights that can help improve customer retention strategies.



This project focuses on \*\*data analysis and business insights\*\*. In the future, it will be extended into a complete \*\*Customer Churn Prediction\*\* project using Machine Learning.





\## 🎯 **Business Problem**



The company is experiencing customer churn and wants to understand:



\- Why are customers leaving?

\- Which customers are most likely to churn?

\- Which business factors contribute the most to churn?

\- What strategies can improve customer retention?



The goal of this project is to answer these questions using data-driven analysis and provide actionable recommendations.





\## 📌 **Project Objectives**



\- Perform exploratory data analysis (EDA).

\- Analyze customer behavior using SQL and Python.

\- Identify the key factors influencing customer churn.

\## 📊 **Business Questions**



This project answers the following business questions:



1\. Does higher monthly charge increase customer churn?

2\. Does contract type influence customer churn?

3\. Does customer tenure affect churn?

4\. Do customers using multiple services churn less?

5\. Which payment method has the highest churn?

6\. How do contract type and payment method together influence customer churn?



\## 📈 **Key Business Insights**



\- Customers with Month-to-Month contracts have the highest churn rate.

\- Customers with Two-Year contracts show the highest customer retention.

\- Customers paying higher monthly charges are more likely to churn.

\- Customers with shorter tenure exhibit higher churn.

\- Customers using fewer additional services tend to churn more frequently.

\- Electronic Check users have the highest churn rate.

\- Month-to-Month customers using Electronic Check represent the highest-risk customer segment.





\## 💼 **Business Recommendations**



Based on the analysis, the following recommendations are proposed:



\- Encourage customers to switch to long-term contracts.

\- Promote automatic payment methods.

\- Improve onboarding for new customers.

\- Increase adoption of bundled services.

\- Design targeted retention campaigns for high-risk customer segments.
